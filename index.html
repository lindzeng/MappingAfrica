Mapping Africa: Mapping Interface



Press the SHIFT key to toggle the between the DRAW and DELETE tool. Press the 's' key to use your mouse normally. Alternatively, you can also use the buttons in the sidebar.

Note: This interface, built with JavaScript, OpenLayers 3, and jQuery, is a demonstration of the tools only. The layer switcher for satellite imagery is obtained by connecting to GeoServer and PostgreSQL servers (not available through GitHub Pages).
